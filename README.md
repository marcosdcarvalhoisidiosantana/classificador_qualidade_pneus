"# classificador_qualidade_pneus" 

Repositório para armazenar modelo de IA utilizando a Teachable Machine da Google para treinar modelos para classificação de imagens.

https://teachablemachine.withgoogle.com/

Base de dados para treinamento: 
https://www.kaggle.com/datasets/warcoder/tyre-quality-classification

Projeto desenvolvido para a criação de portifolio demonstrando o conhecimento adquirido durante o curso de pós graduação da UniPDS - Engenharia IA aplicada. (https://github.com/unipds-engenharia-de-ia-aplicada)

Um agradecimento em especial aos professores e mentores do curso.
